<h1> :hamburger: | Hamburgueria Dev Club </h1>

<p>Este site foi desenvolvido junto ao meu mentor do curso DEVCLUB com o objetivo de finalizar o primeiro módulo de Java Script. Foram utilizados conceitos básicos de HTML e CSS, pois o foco principal do site era desenvolver habilidades de Java Script como map, reduce e filter. Eu modifiquei o layout do site para que eu pudesse testar outras ferramentas de css e deixasse o site mais parecido com a essência do que eu gostaria. </p>

<h1>Acesse o site</h1>
<p>https://hamburgueriadevclub.netlify.app/</p>

<h1>Funcionalidades</h1>

<ul>
  <li>Ao clicar em CONHEÇA NOSSO CARDÁPIO irá aparecer todas as opções da hamburgueria </li>
  <li>Ao clicar em CONFIRA NOSSOS DESCONTOS o site fará uma redução automática de valores, mostrando como ficam os preços com desconto fixo em todos produtos</li>
  <li>Ao clicar em BURGUER PRA GALERA o site mostrará o valor total da compra caso o cliente decida levar uma opção de cada</li>
  <li>Ao clicar em CONFIRA AQUI HAMBURGUER VEGANO o site mostrará apenas as opções veganas da hamburgueria</li>
</ul>


<h1>Como utilizar</h1>
<p>Não há recomendações específicas, apenas clicar no primeiro botão para aparecer as opções.</p>
